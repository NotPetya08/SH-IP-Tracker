# SH-IP-Tracker 🛰️
A high-precision IP tracking tool developed for cybersecurity research.

## Features
- Real-time Geolocation fetching.
- ISP and Organization detection.
- VPN/Proxy and Mobile data checks.
- Easy execution via Bash wrapper.

## Installation
```bash
git clone [https://github.com/NotPetya08/SH-IP-Tracker.git](https://github.com/NotPetya08/SH-IP-Tracker.git)
cd SH-IP-Tracker
chmod +x run.sh
./run.sh
